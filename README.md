# py-marshal

A Rust port of the Python 2.7 [marshal](https://github.com/python/cpython/blob/master/Python/marshal.c) format.

Reads and writes the Python 2.7 marshal serialization. Python 3 marshal opcodes
(interned-string refs, small-tuple, ASCII strings) are out of scope.

## Development

The crate pins Rust 1.87 via `rust-toolchain.toml`. Run the checks with:

```sh
./check.sh
```

A Nix flake is provided: `nix develop` for a shell, `nix flake check` for fmt,
clippy, and tests.

### Test fixtures

`tests/fixtures/*.bin` are marshal blobs generated by real Python 2.7. They are
checked in, so the tests need no interpreter. To regenerate them:

```sh
python2 scripts/gen_fixtures.py
```

Snapshot assertions use [insta](https://insta.rs); review pending snapshots with
`cargo insta review`.
